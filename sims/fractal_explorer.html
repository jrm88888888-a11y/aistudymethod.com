How to read this. The Mandelbrot set on the left plots the parameter c: a point is coloured black if iterating z ← z² + c from z₀ = 0 stays bounded. Coloured points escape, and the colour encodes how quickly. Each c in the Mandelbrot set has a corresponding Julia set — click anywhere on the left to fix c and redraw the Julia set on the right. Points inside the Mandelbrot give connected Julia sets; points outside give Cantor-dust Julia sets.
